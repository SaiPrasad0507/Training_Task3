# Functions & Code Reusability in Python

## Description

This project is based on Python Functions and Code Reusability concepts. It demonstrates how functions help in creating modular, reusable, and efficient programs.

The assignment includes both theory explanations and practical implementations using Python.

---

# Concepts Covered

* Function definition
* Parameters and arguments
* Return statements
* Default arguments
* Variable-length arguments (`*args`)
* Lambda functions
* Recursion
* Variable scope
* Helper functions

---

# Programs Included

* BMI Calculator
* EMI Calculator
* Recursive Factorial
* Fibonacci Series Generator
* Prime Number Checker
* List operations using `map()`, `filter()`, and `reduce()`

---

# Tools Used

* Python
* Google Colab / Jupyter Notebook

---

# Objective

The objective of this assignment is to understand how functions improve:

* Code reusability
* Readability
* Maintainability
* Program structure

---

# Conclusion

This assignment helped in learning how to build reusable and modular Python programs using functions and recursion concepts.
